# React Todo App

A simple and interactive Todo Application built with React and Vite. This project was created to practice React fundamentals, state management, and user interaction handling while building a real-world application.

## Live Demo

https://muhammadikram23.github.io/react-todo-app/

## Project Overview

This Todo App allows users to manage their tasks efficiently by adding, editing, and deleting tasks in real time. The project focuses on implementing core React concepts such as state management, event handling, and dynamic rendering.

## Learning Objective

The primary purpose of this project was to strengthen my understanding of:

* React Functional Components
* React Hooks (`useState`)
* State Management
* Event Handling
* Conditional Rendering
* Component-Based Architecture
* Git and GitHub Workflow
* GitHub Pages Deployment

## Note on UI Design

The styling and visual design of this application were created using AI-assisted design suggestions. This approach allowed me to focus primarily on learning React concepts, application logic, and state management.

While the UI was enhanced with AI-assisted styling, the application functionality and React implementation were developed as part of my learning process.

## Installation

Clone the repository:

```bash
git clone https://github.com/muhammadikram23/react-todo-app.git
```

Navigate to the project directory:

```bash
cd react-todo-app
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

## Build for Production

```bash
npm run build
```
